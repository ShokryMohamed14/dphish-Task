# DPhish Data Table Task

A high-performance, reusable Data Table component built with Vue 3 and TypeScript.

## Features Included
- **Client-Side Pagination:** Easily page through large datasets.
- **Global Search:** Fast debounced search across all visible columns.
- **Dynamic Sorting:** Click column headers to sort ascending or descending.
- **Column Visibility:** Toggle which columns are displayed.
- **Drag & Drop Columns:** Reorder column headers on the fly.
- **Virtual Scrolling:** Render thousands of rows (e.g., 1,500+) smoothly without DOM lag.
- **Responsive & Themed:** Clean UI with empty states and loading skeletons.

## Tech Stack
- **Framework:** Vue 3 (Composition API & `<script setup>`)
- **Language:** TypeScript
- **Build Tool:** Vite
- **Utilities:** lodash-es (for debounce)

## Setup Instructions

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
   *The production-ready files will be generated in the `dist/` directory.*
